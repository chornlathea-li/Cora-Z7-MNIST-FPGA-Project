# Cora-Z7-MNIST-FPGA-Project
This project demonstrates MNIST handwritten digit recognition on the Cora-Z7 FPGA using a lightweight fully-connected neural network with the structure 100 → 16 → 8 → 10.

## Hardware

- **FPGA Board:** Digilent Cora-Z7  
- **FPGA Chip:** Zynq-7007S
  
## Environment

- **Vitis HLS, Vivado, Xilinx Vitis:** 2021.1  
- **OS Tested:** Windows 11
